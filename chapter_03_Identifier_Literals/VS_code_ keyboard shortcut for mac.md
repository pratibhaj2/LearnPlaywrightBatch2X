# Visual Studio Code Keyboard Shortcuts

## macOS Default Shortcuts

> Press **Cmd+K Cmd+S** in VS Code to open the Keyboard Shortcuts editor and customize any shortcut.

---

## General

| Shortcut | Action |
|----------|--------|
| `Cmd+Shift+P` | Show Command Palette |
| `Cmd+P` | Quick Open / Go to File |
| `Cmd+Shift+N` | New Window |
| `Cmd+W` | Close Window |
| `Cmd+,` | Open User Settings |
| `Cmd+K Cmd+S` | Open Keyboard Shortcuts |
| `Cmd+Shift+T` | Reopen Closed Editor |
| `Ctrl+Cmd+F` | Toggle Full Screen |
| `Cmd+K Z` | Toggle Zen Mode |
| `Cmd+O` | Open File |
| `Cmd+S` | Save |
| `Cmd+Shift+S` | Save As |
| `Cmd+K S` | Save All |
| `Cmd+W` | Close File |
| `Cmd+K W` | Close All |
| `Cmd+Q` | Quit |
